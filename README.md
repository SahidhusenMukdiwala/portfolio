# portfolio
Sahidhusen Mukdiwala - Professional Portfolio &amp; Personal Website

Welcome to the source code of my personal portfolio website, a high-performance, modern web application designed to showcase my expertise as a Full Stack Developer. Built from the ground up with a focus on seamless user experience, stunning visual design, and robust architecture, this repository serves as both a digital resume and a testament to my technical capabilities in the MERN stack and modern frontend ecosystems.

Technical Architecture & Stack This project is engineered using Next.js 15 (App Router) and TypeScript, ensuring type safety, server-side rendering benefits, and exceptional SEO performance. The user interface is crafted with Tailwind CSS, utilizing a highly customized, dual-theme design system that supports seamless toggling between an immersive Dark Mode and a clean Light Mode. To elevate the user experience, I integrated Framer Motion and Lenis to create buttery-smooth scroll animations, dynamic page transitions, and engaging micro-interactions (such as parallax effects and infinite marquee sliders).

Key Features & Implementation

1. Dynamic Theming: A robust CSS-variable foundation that manages comprehensive color palettes for dark and light modes without visual flickering.
1. Performance Optimized: Achieves near-perfect Lighthouse scores through Next.js static generation, optimized asset loading, and modular component architecture.
3. Interactive UI Components: Features custom implementations like an infinite skill slider, interactive project grids, a visual experience timeline, and a dynamic particle background.
4. Integrated Communications: Contains a fully functional, secure contact form powered by Nodemailer, utilizing server-less API routes to handle direct email delivery seamlessly.
5. Responsive Design: A mobile-first approach ensuring the portfolio looks and functions flawlessly across all device breakpoints.
